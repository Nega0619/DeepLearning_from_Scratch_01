# Affine 변환의 오차역전파

W^T에서 T는 전치행렬을 의미.

> 전치행렬이란, W의 (i, j)위치 원소를 (j, i)위치로 바꾼 것

- np.dot의 미분은 W의 전치행렬이 된다. (왜 이렇게 되는지 증명할 순 없는 수준이지만, 일단 외우고 넘어가자!)
- 전치행렬이 되는 위치도 중요하다! X\*W에서 X에 대한 미분이면 dX * W_T, W에 대한 미분이면 X_T * dW이다!
- 미분 전의 행렬의 shape과 미분 후의 행렬 shape은 동일해야 한다!
